这里只保存跨 MOD 可复用的规则。不要在这里写某个 MOD 的细节、测试流水账或临时待办。

- `04_联机与同步规范.md`：房主/客户端、Photon、RPC、房间属性和中途加入同步原则。
- `05_发布与版本规范.md`：版本号、发布前检查和文档风格。
