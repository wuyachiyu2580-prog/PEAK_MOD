# 06 UI 与字体规范

更新时间：2026-05-08

本规范为所有 MOD 通用。任何新建 `TextMeshProUGUI` / `TextMeshPro` / `TMP_Text` 的地方都必须遵守。

## 背景：CJK 乱码的根因

`gameObject.AddComponent<TextMeshProUGUI>()` 默认使用 `TMP_Settings.defaultFontAsset`，该 FontAsset 只烘焙了 ASCII 字形。中文、日文等 CJK 字符会渲染成豆腐块（□□□）或空白。

## 铁律

1. **绝对不要**让 `tmp.font` 保持默认值。`AddComponent` 后必须立即指派一个含 CJK 的 `TMP_FontAsset`。
2. **绝对不要**直接写 `tmp.font = TMP_Settings.defaultFontAsset;`（等价于不赋值）。
3. **绝对不要**从字体访问器里返回 `null`。返 null 会让 TMP fallback 到系统字体，描边失效且字变粗，比豆腐块更难看。
4. **不要自己打 SDF 图集**。PEAK 游戏本体已经烘好了含完整中文的 FontAsset，直接复用即可。

## 四级兜底顺序（从高到低）

所有 MOD 的 `FontHelper.GetChineseCapable()` 必须按以下顺序取：

1. `GUIManager.instance.GetComponentInChildren<AscentUI>(true).text.font`
   - 原版主 UI 里的中文文本组件，字形覆盖最广。
2. `GUIManager.instance.heroDayText.font`
   - "第 N 天"字样的字体，含完整中文。
3. `UnityEngine.Object.FindAnyObjectByType<TextMeshProUGUI>().font`
   - 兜底扫全场，拿任意一个可用字体。
4. `TMP_Settings.defaultFontAsset`
   - 最后兜底，宁肯没中文也别返 null。

## 标准实现

每个 MOD 在自己的 `Helpers/FontHelper.cs` 下建一个 `internal static class FontHelper`，对外只暴露：

```csharp
public static TMP_FontAsset GetChineseCapable();
public static void InvalidateCache();   // 场景重载时调用
```

内部必须有一个 `private static TMP_FontAsset _cached;`，避免每次新建 TMP 都全场 `FindAnyObjectByType` 卡帧。

**参考实现**：
- `MOD开发\PlayersInfo\PlayersInfo\Helpers\FontHelper.cs`
- `MOD开发\ItemInfoCN\ItemInfoCN\Helpers\FontHelper.cs`

两份保持同源同结构。新写 MOD 直接复制。

## 场景就绪判据

`GUIManager.instance` 在 Awake 阶段还是 null，必须等 HUD 挂载完才能取字体。建议做法：

- 先判定场景依赖就绪（如 `GameObject.Find("GAME/GUIManager")` 或 `GUIManager.instance != null`）。
- 再调 `FontHelper.GetChineseCapable()`。
- 全部实例化完成后，最后一步才赋值给 static 字段，保证"要么全有要么全没"不留孤儿。

参考：`ItemInfoCN/Plugin.cs` 的 `AddDisplayObject` 方法。

## 描边

描边统一走各 MOD 自己的 `TmpOutlineHelper.Apply(tmp, width, color)`，默认宽度 0.08f。不要在各自 TMP 上手写 `outlineWidth = ...` 导致风格不一。

## 字号

- HUD 文本字号建议用 `ConfigEntry<float>` 暴露给用户，范围 8-24。
- 经验默认值：名字/状态条 13-18，物品信息 20，标题 24。
- 不要硬编码超过 24 的字号。

## 禁止回退

- 禁止让 `tmp.font` 为 null 或保持 defaultFontAsset。
- 禁止在访问器里返回 null。
- 禁止复写"拿字体"的自造实现。统一走各 MOD 的 `FontHelper`。
- 新增 MOD 忘记加 FontHelper → 直接复制现成的两份之一。
